
## 📌 기능 목록

- 회원 기능 
  - 회원 등록
  - 회원 탈퇴
  - 로그인
  - 로그아웃
  

- 게시물 기능
  - 게시물 생성
    - 텍스트 저장 
    - 이미지 저장
    - 동영상 저장 
  - 게시물 조회
    - 정렬
    - 검색
  - 게시물 삭제
  - 게시물 수정


- 댓글 기능
  - 댓글 생성 
    - 대댓글 생성
  - 댓글 조회
  - 댓글 삭제 (=숨김처리) 
    - 삭제시 대댓글은 남아있음
  - 댓글 수정


- 좋아요 기능
  - 좋아요 체크 
  - 좋아요 해제
  - 좋아요 체크한 게시물 보기(좋아요 체크 시간순)


- 북마크 기능 
    - 북마크 체크
    - 북마크 해제
    - 북마크 체크한 게시물 보기(북마크 체크 시간순)
